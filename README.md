# JavaScript Build Tool

이번 예제에 사용하는 도구에는 * 표시

1. 의존성 관리자

  프로젝트에서 사용하는 라이브러리를 관리

  예시:
    - npm*
    - bower
    - jspm

1. 트랜스파일러

  소스 코드를 브라우저가 이해할 수 있는 JavaScript로 번역 (브라우저를 위한 컴파일러)

  예시:
    - babel*
    - coffee script
    - traceur
    - closure compiler

1. 작업 자동화 도구

  반복적인 작업을 자동화

  예시:
    - gulp
    - grunt
    - npm script*
    - git hook

1. 모듈 번들러

  여러 모듈 파일을 브라우저에서 실행 가능하도록 묶어줌

  예시:
    - browserify
    - webpack*
    - rollup
